## Installing a Minecraft Modpack

### Using Twitch / Overwolf Client

This is the easiest way to install a modpack. [Download Overwolf](https://download.overwolf.com/install/Download?Channel=web_dl_btn2), then navigate to `Mods > Minecraft` and search for `All the Mods 6`. Install the mod and make sure to set your client java parameters to add some move available ram.
